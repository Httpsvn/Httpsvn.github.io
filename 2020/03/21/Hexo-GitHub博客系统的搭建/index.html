Hexo+GitHub博客系统的搭建

如果单是作为笔记功能，那么由Hexo提供的页面框架，GitHub提供网络访问环境。Hexo+GitHub搭建的博客系统无疑是最好的选择。

环境搭建

  1. 安装git,通过git --version,检查是否安装成功
  2. 安装nodejs,通过node -v、npm -v,检查是否安装成功
  3. 安装hexo

如果安装时出现了一下警告

是因为mac下需要 fsevents，在windows或linux环境下，请忽略这个错误。
